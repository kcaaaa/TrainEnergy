# 中国高铁站节能降耗演示系统

## 项目介绍
这是一个基于Vue 3 + Element Plus开发的高铁站节能降耗演示系统，用于展示高铁站能源管理、节能分析和系统管理功能。

## 核心功能

### 1. 能源管理
- 实时监控高铁站各区域能耗数据
- 能耗数据分析与统计
- 节能策略推荐

### 2. 节能分析
- 能耗趋势分析
- 设备能耗对比
- 节能效果评估

### 3. 系统管理
- **系统概览**：展示系统运行状态和关键指标
- **用户管理**：用户账号的增删改查和权限控制
- **角色管理**：角色定义和权限分配
- **菜单管理**：系统菜单的配置和管理
- **系统日志**：操作日志和系统日志的查询
- **系统设置**：系统参数配置、安全设置、通知设置和备份管理

## 技术栈
- **前端框架**：Vue 3 + Vite
- **UI组件库**：Element Plus
- **图标库**：FontAwesome
- **数据可视化**：ECharts
- **路由管理**：Vue Router
- **状态管理**：Vue 3 Composition API

## 安装与运行

### 环境要求
- Node.js ≥ 16.0.0

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 项目结构
```
src/
├── assets/           # 静态资源
├── components/       # Vue组件
│   ├── EnergyManager/       # 能源管理组件
│   ├── EnergyAnalysis/      # 节能分析组件
│   └── SystemManagement/    # 系统管理组件
│       ├── SystemOverview.vue      # 系统概览
│       ├── UserManagement.vue      # 用户管理
│       ├── RoleManagement.vue      # 角色管理
│       ├── MenuManagement.vue      # 菜单管理
│       ├── LogManagement.vue       # 系统日志
│       └── SystemSettings.vue      # 系统设置
├── router/           # 路由配置
├── views/            # 页面视图
├── App.vue           # 根组件
└── main.js           # 入口文件
```

## 使用说明

### 登录系统
1. 访问系统登录页面
2. 使用默认账号登录：
   - 用户名：admin
   - 密码：12345678

### 系统管理模块
登录后，通过左侧导航菜单点击"系统管理"进入系统管理模块，即可使用以下功能：

#### 系统概览
- 查看系统运行状态
- 监控在线用户数量
- 查看系统资源使用情况

#### 用户管理
- 创建新用户账号
- 编辑用户信息
- 设置用户角色和状态
- 删除用户账号

#### 角色管理
- 创建和管理角色
- 为角色分配权限
- 编辑角色信息

#### 菜单管理
- 配置系统导航菜单
- 设置菜单权限
- 调整菜单顺序

#### 系统日志
- 查询操作日志
- 查看系统日志
- 筛选和导出日志

#### 系统设置
- 配置系统基本信息
- 设置安全策略
- 配置邮件服务器
- 管理系统备份
